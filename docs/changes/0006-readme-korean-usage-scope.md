# Change 0006: Clarify Korean README usage and scope

## Summary

Rewrote the README in clearer Korean so installation, download, authentication, current capabilities, and clean-room scope are explicit.

## Why

The repository already had working commands and release downloads, but the documentation did not clearly separate what is implemented today from the larger public Claude Code feature surface. That made the product easier to misunderstand than to use.

## Outcome

- `README.md` now gives a concrete Korean quick-start with install, download, auth, and command examples
- the README now states that the product surface is terminal-first rather than a web UI
- the README now explicitly distinguishes current implementation scope from public-Claude-inspired future clean-room targets
